## 架构
- app 端主要技术：JWT + Redis
- admin 端（管理平台）主要技术：Thymeleaf + LayUI + Shiro + Redis

## 结构
- ums - 用户管理
- sys - 系统管理

## api分布
- /app/**   app   接口地址

- /admin/** admin 接口地址
